# Beginner Example

A single-file HTML portfolio generated from `portfolio-prompt.md` with minimal edits.

## What's here

## Run locally
```bash
npx serve .
# or
python -m http.server 8000
```

## What was changed from raw AI output
1. Tightened the hero copy (cut "passionate" and "innovative")
2. Replaced em dashes with hyphens
3. Added `:focus-visible` styles for keyboard navigation
4. Added `prefers-color-scheme` dark mode
